# airbnb_ui

This a flutter travel app ui inspired by Airbnb .

## Getting Started

```shell
git clone https://github.com/Dera-codes/Airbnb_Ui.git
```
```shell
flutter clean
```
```shell
flutter packages get
```
```shell
flutter run
```
